Program description

A program made to encrypt text. Its true value is that you are able to make your own encrypted text.
Make your own encrypted text out of a text file and save it inside a safe spot.
Without the .txt file with the encrypted pattern, it is impossible to unencrypt your text, so be careful.

To learn how to make your own encrypted patterns, please, read the User Manual here.

Program features
  • Is there a letter not encrypted? No problem, add it inside the .txt file as stated in the user manual
  • No matter what kind of alphabet you are using, the program should work.
  • Possibility to save all the encryption patterns you want.
  • Encrypt any kind of text you want.
  • Feather size.
