Program description

This program will shut down any Windows PC after any mouse movement.

Before using this program, make sure you understand what this program does by reading its guidelines.
I am not responsible for any data loss due to undesired shutdown by you or someone else by the utilization of this program.

Program features
  • Command prompt "Shutdown -s -t 0" to shut down.
  • Timer before being aware if you move your mouse.
  • Shutdowns your PC after you move your mouse.
  • No need to install any other further program.
  • Portable .exe (no installation needed)
  • Extremely easy to use.
