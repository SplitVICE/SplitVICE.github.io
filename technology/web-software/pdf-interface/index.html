Program description

Visualize PDF files stored in a directory on a website. Access and view your PDF files by writing the webserver IP on your PC, laptop or mobile devices.

Visit PDF Interface's Github website to read how to configure this web application and to run it on your webserver clicking here.

PDF Interface allows you to set a password so files can only be viewed by typing the password.
