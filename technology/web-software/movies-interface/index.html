Program description
Tiny web application to stream videos stored in your computer on the local network.
Movies Interface has been tested with XAMPP, Docker PHP containers, Apache 2, and Nginx; either one should be enough to run the backend.

Note
Movies Interface uses HTML5 web player which means only the following formats are supported:
  • mp4
  • WebM
  • ogg
  • mov
  • mkv (no sound)
Go to Movies Interface's Github repository where a small guide to setup Movies Interface with XAMPP and Docker is mentioned.
Application features
  • Feather size.
  • Relatively easy to configure.
  • Light local network bandwidth consumption.
  • Compatible with most browsers.
  • Responsive to screen size (can be used on mobile devices).
  • Direct links to download movies you add to the software.
