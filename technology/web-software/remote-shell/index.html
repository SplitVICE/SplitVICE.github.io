Program description

Control your or someone elses PC with terminal commands remotely.
Available default actions to perform with buttons. Currently the following:

  • Turn off PC
  • Restart PC
Check the Github repository to read instructions about how to install and execute on your PC.

Program features
  • Can be used to control on mobile devices (turn off your PC with phone).
  • Dark mode so your eyes do not get burn.
  • Easy to install.
