Web application description

Free file hosting web application you can run on your own local network or upload it into any web host on the Internet.
Share uploaded files and have a list of uploaded files (or hide uploads on the site).

Live demo here (might take up 10 seconds to load due to Heroku sleep policy)

Note: I learned Vue with this application so this is another app I appreciate
Program features
  • Host your own files up free storage space you have in your computer
  • Share the files with an URL generated by the web application
  • Limit file size or leave it unlimited
  • Show all files hosted on a public list of hide the public list
  • Optimized to be as fast as it can
